Submission category · Proceedings

Talks

20-minute presentation with a 1–2 page extended abstract. Proceedings.

What it is

A 20-minute presentation backed by a 1–2 page extended abstract, modelled on SIGGRAPH Talks. Ideal for practitioners and for sharing a production result, pipeline or lesson learned without writing a full paper. Open to contributors without an academic degree.

Requirements

  • 1–2 page extended abstract in the acmart sigconf format.
  • A 20-minute talk; the abstract is what enters the proceedings.
  • ORCID for each author; no academic degree required.

What ACM says

ACM does not define a central 'talk' format; the established model is the SIGGRAPH Talk, a short extended abstract published in the ACM DL alongside the presentation. We follow that model, so your talk gets a DOI via TAPS. Showcases that name a specific stage are reviewed single-blind.

Template & formatting

acmart · sigconf — sample

The acmart class sets the typography for you, so don't change fonts, font size, margins or line spacing by hand. The sigconf format is two-column on US-Letter and uses the Libertine font set, which the class loads automatically. Citations use the bundled ACM-Reference-Format style. For the review version, add the anonymous option: \documentclass[sigconf,anonymous]{acmart}. Stay within the page limit above (it excludes the reference list).

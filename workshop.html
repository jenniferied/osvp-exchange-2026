Submission category · Proceedings

Workshops

Hands-on session at the LED wall (60–90 min) with a short proceedings write-up. Proceedings.

What it is

A 60–90 minute hands-on session at the LED wall — a tutorial, a tool walkthrough or a guided exercise — accompanied by a short proceedings write-up so the material stays citable after the event.

Requirements

  • A workshop proposal (goals, audience, what happens at the LED wall, duration).
  • A short write-up (a few pages) in the acmart sigconf format for the proceedings.
  • ORCID for each organizer.

What ACM says

The write-up follows the same acmart / ICPS / TAPS pipeline and receives a DOI. Proposals are curated by the program committee rather than double-blind reviewed.

Template & formatting

acmart · sigconf — sample

The acmart class sets the typography for you, so don't change fonts, font size, margins or line spacing by hand. The sigconf format is two-column on US-Letter and uses the Libertine font set, which the class loads automatically. Citations use the bundled ACM-Reference-Format style. For the review version, add the anonymous option: \documentclass[sigconf,anonymous]{acmart}. Stay within the page limit above (it excludes the reference list).
